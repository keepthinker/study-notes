## 使用快捷键
Ctrl + C：复制选中的单元格
Ctrl + V：粘贴剪贴板上的内容
Ctrl + Z：撤销上一步操作
Ctrl + Y：重做上一步被撤销的操作
Ctrl + S：保存当前工作簿
Ctrl + F：在当前工作表中进行查找
Ctrl + H：在当前工作表中进行替换
Alt + =：自动求选中范围的总和
Ctrl + Shift + Arrow key：选择连续的单元格范围

## 函数
- Alt + =：自动求选中范围的总和
- 求和： =SUM(C2:C12)-SUM(D3:D12)
- 判断：=IF(C2=D2,"C列和D列值相同","C列和D列值不同")，=IF(D6="a","a","not a")
- 平均值： AVERAGE 函数可以计算一个范围的单元格的平均值， =AVERAGE(B1:B10)
- COUNT：COUNT函数可以计算一个范围的单元格中含有数值的单元格的数量。例如，=COUNT(C1:C10) 将返回C1 至C10 单元格中含有数值的单元格的数量。
- 将多个文本字符串连接：这时候你可以使用CONCATENATE函数。CONCATENATE函数用于将两个或更多的文字字符串连接为一个文字字符串。例如，=CONCATENATE(A1, " ", B1)将结合A1 和B1 单元格的内容，并在中间插入一个空格。
- 查找相关数据时，VLOOKUP函数将是你的好帮手。例如，=VLOOKUP("John", A1:B10, 2, FALSE)将在A1:B10 的范围中查找“John”，并返回与“John” 在同一行的第2 列的数据。
- 基于一系列条件来进行计算：这时候你可以使用SUMIFS函数。SUMIFS可以基于一个或多个给定条件来对范围的数据进行求和。例如，=SUMIFS(A1:A10, B1:B10, ">20", C1:C10, "<50")将会计算在B1:B10范围内大于20且C1:C10范围内小于50的对应A1:A10的数据总和。
- 在列表或数据集中找到最大值或最小值: 你可以使用MAX和MIN函数。例如，=MAX(A1:A10)将返回范围A1:A10 中的最大值，而=MIN(A1:A10)则会返回这个范围内的最小值。
- 对日期进行操作: Excel 有许多专门用于处理日期的函数，如YEAR、MONTH、DAY、TODAY等。例如，=YEAR(A1)可以返回A1 单元格中日期的年份；=TODAY()函数则返回当前日期。
 
### SUMIF 和SUMIFS 函数
目的：基于单一条件或多重条件计算销售总额。
公式：
单一条件：=SUMIF(D2:D100, "北部", E2:E100)    // 区域，条件，[求和区域]
多重条件：=SUMIFS(E2:E100, D2:D100, "北部", B2:B100, 1001) //[求和区域], 区域1, 条件1, 区域2, 条件2, ...

## 自动填充
Excel 的自动填充功能可以让你迅速填充一列或一行的数据。例如，我们想在A 列的1 到10 单元格中填充数字1至10，只需在A1 单元格输入1，然后将滑鼠放到A1 单元格右下角，出现黑色十字后，直接向下拖拽至A10 单元格。

## 多单元格输入
1.选中需要输入相同内容的单元格，然后按住“Ctrl”的同时用鼠标左键点击多个单元格。 
2.然后在最后一个选中的单元格里输入内容，输入内容后再次点击Ctrl键+Enter键。

## 使用条件格式化
条件格式化可以根据单元格的值改变单元格的格式，如字体颜色、背景色、数字格式等。这可以帮助你快速识别出符合特定条件的数据。
### 高亮显示重复值
选中数字区域，之后点击开始——条件格式——突出显示单元格规则——重复即可。

## 快速复制上一行内容
选中下一行中对应的空白单元格区域，按【Ctrl+D】即可完成快速复制。

## 快速选中公式所需的某列数据区域
设置公式时，我们经常直接用鼠标往下拖拉选中数据区域，其实，只要借助【Ctrl+Shift+↓】组合键就可以快速在公式中输入数据区域。

## 使用数据验证
数据验证是一种功能，允许你限制在特定单元格中可以输入的数据类型，或者从预定义的列表中选择数据。这可以帮助你防止输入错误的数据。
